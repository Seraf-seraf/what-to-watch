Введение

Данный проект представляет собой REST API для управления базой данных фильмов, включая функции оставления комментариев, регистрации пользователей и управления списком избранных фильмов. Сервис интегрируется с КиноПоиском, что позволяет получать комментарии к фильмам и предоставляет пользователям возможность активно взаимодействовать с ними.

Базовый URL: https://localhost

Comment

Пользователь может оставлять отзывы к фильмам и отвечать на другие комментарии. Администратор может удалять, обновлять комментарии. Удалить можно только те, на которых нет ответов. Рейтинг фильмов рассчитывается исходя из оставленных оценок в отзывах. Ежедневно комментарии к фильмам загружаются с сервиса кинопоиск.
